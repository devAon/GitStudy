최초 1회 실행
git config --global user.name "your_name"

git config --global user.email "your_email@example.com"

git config --list

--------------------------------------------------------
git init 현재 디렉토리를 로컬저장소로 설정

//로컬저장소로 설정할 프로젝트 위치로 이동
cd C:/

//로컬 저장소로 설정
git init

//만약 init 취소하려면 아래의 명령어 입력
rm -r .git

--------------------------------------------------------
git status 로컬저장소의 현재 상태를 보여줌


git add 파일을 준비영역으로 옮김
git add a.html
git add .
git add -i

git commit 준비영역의 파일을 로컬 저장소에 저장.
git commit 
git commit -m "커밋 메시지"
git commit -a -m "커밋 메시지"
--------------------------------------------------------
git log 로컬저장소의 커밋 이력을 조회

--------------------------------------------------------
git remote  로컬저장소와 원격저장소를 연결
git remote add orgint [자신의 github 원격저장소 주소]
git remote -v

--------------------------------------------------------
git push  원격저장소에 저장
git push -u origin master


--------------------------------------------------------
1) 브랜치 보기
git branch

2) 브랜치 생성
git branch [브랜치명]

3) 브랜치 수정
git branch -m [브랜치명][바꿀이름]

4) 브랜치 삭제
git branch -d [브랜치명]

--------------------------------------------------------
checkout 워킹 디렉터리의 소스를 특정 커밋 또는 특정 브랜치로 변경

1) 특정 브랜치로 워킹 디렉터리 변경
git checkout [브랜치명]

2) 브랜치 생성 및 체크아웃을 같이할 경우
git checkout -b develop

--------------------------------------------------------
merge 다른 두개의 브랜치 소스를 병합
git checkout master
git emrge develop
